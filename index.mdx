0x0 is a small set of AI components for developer tools.

Use `@anonymous-dev/0x0-server` as the local AI runtime. Use
`@anonymous-dev/0x0-nvim-chat` for chat, inline edits, and review flows. Use
`@anonymous-dev/0x0-nvim-completion` for ghost-text completion. Use
`@anonymous-dev/0x0-contracts` when another client needs the same request and event shapes.
