title: 'Brew'

---

The `brew/` module manages all Homebrew dependencies through a single
[Brewfile](https://github.com/Homebrew/homebrew-bundle).

## Installation

```bash
./brew/install-brew.sh
```

This script:
1. Installs Homebrew if it's not already present.
2. Runs `brew bundle` against the Brewfile to install everything.

## Taps

| `oven-sh/bun` | Bun JavaScript runtime |
